# Algorithm
BaekJoon Online Judge, Codeforces Problem Solving (by C++, Java)

BOJ ID : wldndsla8

- 사각형의 면적(겹치는 부분 포함)

좌표를 표현할 때 x좌표가 열, y좌표가 행임(매번 헷갈림)

사각형이기 때문에 경계 전까지는 사각형 형태를 유지함

이차원 배열을 통해 좌표평면을 구현하고 각 인덱스 값이 가로+1 세로+1 넓이 1의 사각형이 존재하는지

따라서 각(가로, 세로 길이만큼) 좌표를 탐색하며(경계값 제외) 해당 인덱스 값이 존재 안한다면 새롭게 만들어지는 넓이 1만큼의 사각형을 총 넓이에 더해준다. 이후 존재상태로 변경(인덱스값 변경)
