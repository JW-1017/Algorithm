# Algorithm
BaekJoon Online Judge, Codeforces Problem Solving (by C++, Java)

BOJ ID : wldndsla8

- 동전교환

- 중복을 허용하는 조합
  금액(1부터 목표 값)을 변경해가며
  간격을 반복문을 돌며 해당 값에 대한 값을 갱신

- 중복을 허용하지 않는 조합
  순차적으로 경우의 수 계산
  간격(동전 값)이 우선으로 반복문을 돌면서
  목표 값까지 반복문을 돌면서 차이가 나는 곳(간격의 차이)의 값을 더하며 갱신
  (1부터 경우의 수 목표치까지 계산해나감)

  1
  1 + 1, 2 (dp[1] 에서 1더한값)
  1 + 1 + 1, 1 + 2 (dp[2] 에서 1더한값 (1 + 1 + 1), dp[1] 에서 2더한 값 (1+2))
